| Input | Action |
| ----- | ------ |
| W     | UP     |
| A     | LEFT   |
| S     | DOWN   |
| D     | RIGHT  |
| SPACE | SHOOT  |

| Input       | Action |
| ----------- | ------ |

| Mouse DOWN  | DOWN   |
| Mouse LEFT  | LEFT   |
| Mouse RIGHT | RIGHT  |
